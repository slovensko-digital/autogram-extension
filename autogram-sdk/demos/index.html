Autogram SDK demos

One demo per build flavor in dist/:

  • ESM, bundled deps — dist/bundled/*.mjs, self-contained modules for static hosting without a bundler.
  • ESM, externalized deps — dist/*.mjs, the npm package build; bare imports resolved here via an import map to esm.sh (needs network).
  • IIFE, CombinedClient — dist/index-all.iife.js via classic <script> tag, AutogramSDK global.
  • IIFE, DesktopClient (custom UI) — headless desktop signing with custom progress UI.
