Autogram v mobile integration demo

Example AVM Integration

This example follows the same lightweight Vite pattern as example-extension-usage, but drives a real AVM integration flow: register or reuse an integration, upload a document, generate a mobile deep-link or QR payload, and wait for the signed file.

Paste the AVM URL containing guid, key, and optionally integration. If the page was opened with those query params already present, use “Load current page URL”.

Complete request

AVM expects a raw signedData value over the server-provided dataToSign. This demo prepares the request and posts the final AVM /documents/{guid}/sign call when you provide the certificate and signed payload.
